# Arsenal FC Champions Parade – 31 May 2026

Interactive map of the Arsenal FC 2025/26 champions parade through Islington, showing road closures, the parade route, closed parks, and nearby transport hubs.

**Live map:** https://angrytinyfemale.github.io/parade_31May/

---

## What's on the map

| Layer | Description |
|---|---|
| 🟡 Parade Route | Official parade route through Islington |
| 🔴 Road Closures | All roads closed to vehicles (4am – ~8pm) |
| 🟠 Pedestrian Closures | Roads closed to pedestrians only |
| 🟩 Closed Parks | Green spaces closed on the day |
| 🔴 Tube / Overground | Nearest stations |
| 🟢 Rail Stations | Nearest National Rail stations |
| 🔵 Bus Stops | Active bus stops within 1.5 km of the closure zone |

Click any feature for details. Partial closures show the specific section affected.
